# odoo_yo

Automation utilities for Odoo Online instance `jesus-chavez-galaviz.odoo.com`.

- Playbooks in `playbooks/`
- Invoice/complements/addenda API bridge in `src/odoo_bridge/`
- App UI theme manager in `src/odoo_bridge/app_ui_unocss/`
- Documentation in `docs/`

Quick commands:

- `uv run odoo-bridge --status`
- `uv run odoo-app-ui --status`
- `uv run odoo-theme --status`
